type token = {
token : (string, [ `Msg of string ]) Stdlib.result;

A token to include in the "Authorization" header, or an error if we failed to get a token.

token is valid until this time. If token is an Error then this is the earliest time to try again. If None, token does not expire.

val v : get_token:(unit -> token Lwt.t) -> string -> t

v ~get_token account is a configuration that uses get_token when it needs to get or refresh the API token. Note: get_token can return a failed token, in which case the expiry time says when to try again. If get_token instead raises an exception, this is turned into an error token with a 1 minute expiry.
